# Loyalty Points Display Extension

A Shopify theme extension that displays customer loyalty points information.

## Features

- Shows current points balance
- Displays loyalty tier
- Shows points that will be earned on current order
- Shows available discount from points redemption

## Installation

This extension can be added to any Shopify theme using the theme editor.

## Usage

The extension automatically fetches and displays loyalty points information for logged-in customers.

## Configuration

Configure the following in the Shopify admin:
- API endpoint URL
- Display style and colors
- Points display format
