# Rewards Announcement Bar

**Type:** Theme App Extension (App Block)
**Shopify Compliance:** 2024-2025 ✓

## Merchant Installation

1. Go to **Online Store → Themes → Customize**
2. Navigate to **Header** section
3. Click **Add Block**
4. Select **"Rewards Announcement"**
5. Enter your **Widget ID**
6. Customize message and CTA text
7. Toggle dismissible option
8. Click **Save**

## Features

- Full-width bar at top of site
- Customizable message and CTA
- Optional dismiss button (remembers user preference)
- Responsive design
- Conditional rendering

## Use Cases

- Promote rewards program enrollment
- Announce special member-only offers
- Drive traffic to membership portal
- Limited-time campaign announcements

## Removal

Remove block in Theme Customizer or uninstall app.
