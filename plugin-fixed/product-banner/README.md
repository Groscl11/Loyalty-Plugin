# Rewards Product Banner

**Type:** Theme App Extension (Section Block)
**Shopify Compliance:** 2024-2025 ✓

## Merchant Installation

1. Go to **Online Store → Themes → Customize**
2. Navigate to a **Product Page**
3. Click **Add Block** or **Add Section**
4. Select **"Rewards Banner"**
5. Enter your **Widget ID**
6. Customize title, description, and button text
7. Click **Save**

## Features

- Appears on product pages
- Customizable title, description, and CTA button
- Responsive design
- Conditional rendering (only shows when rewards available)

## Placement

Can be placed anywhere on product pages:
- Above product form
- Below product description
- In sidebar

## Removal

Remove block in Theme Customizer or uninstall app. No cleanup needed.
