# Floating Rewards Widget

**Type:** Theme App Extension (App Embed)
**Shopify Compliance:** 2024-2025 ✓

## Merchant Installation

1. Go to **Online Store → Themes → Customize**
2. Click **App Embeds** in the left sidebar
3. Find **"Rewards Floating Widget"** and toggle it ON
4. Enter your **Widget ID** from the Rewards dashboard
5. Click **Save**

## Features

- Appears as a floating button in bottom-right corner
- Fully customizable text and styling
- Click-to-claim functionality
- Auto-hides when no rewards available
- Privacy-compliant (respects customer tracking preferences)

## Removal

Toggle OFF in Theme Customizer or uninstall the app. No code cleanup needed.

## Developer Notes

- Widget loads asynchronously
- Calls `/api/widgets/render` endpoint
- Read-only (no cart/checkout mutation)
- App Store compliant
